# 🚀 JS Capstone: Shopping & Admin Management

An e-commerce project consisting of two main modules: a Customer Shopping Interface and a Product Admin Dashboard, built with a focus on Object-Oriented Programming (OOP) and API integration.

<p align="center">🌐 Live Demo: https://js-capstone-shopping-admin.vercel.app/</p>
